-------------------- < SIMPLE Algorithm > --------------------

вҶ’ Semi-Implicit Method for Pressure Linked Equations

The key tricks for the SIMPLE algorithm are to :
    вҶ’ Derive an equation for pressure from the momentum and continuity equations.
    вҶ’ Derive a corrector for the velocity field, so that it satisfies the continuity equation.

Solution process :
    1.  Solve the momentum equation for the velocity field. This velocity field does not satisfy the continuity equation.
        вҶ’ MU = -вҲҮp
        вҶ’ H = AU - MU

    2.  Solve the Poisson equation for the pressure field.
        вҶ’ вҲҮВ·(AвҒ»В№вҲҮp) = вҲҮ(AвҒ»В№H)

    3.  Use the pressure field to correct the velocity field so that is satisfies the continuity equation.
        вҶ’ U = AвҒ»В№H - AвҒ»В№вҲҮp
        
    4.  The velocity field now does not satisfy the momentum equations. So repeat the cycle.

Discretization of the Navier-Stokes equations:
    1.  Governing Equations (Incompressible Navier-Stokes):
            ПҒвҲӮu/вҲӮt + ПҒ(uвӢ…вҲҮ)u = -вҲҮp + ОјвҲҮВІu + ПҒf
                         вҲҮu = 0

    2.  Temporal Discretization (Implicit Euler):
        1st Order :
            вҲӮ/вҲӮt вҲ«ПҒu dV вүҲ ПҒО”V/О”t вӢ… вҖӢ(un+1вҖӢ вҲ’ unвҖӢ)
            вҶ’ AtP = ПҒО”xО”y/О”t
            вҶ’ Qt = ПҒО”xО”y/О”t вӢ… un

        3rd Order :
            вҲӮ/вҲӮt вҲ«ПҒu dV вүҲ ПҒО”V/2О”t вӢ… вҖӢ(3un+1вҖӢ вҲ’ 4un + un-1вҖӢ)
            вҶ’ AtP = 3ПҒО”xО”y/2О”t
            вҶ’ Qt = ПҒО”xО”y/2О”t вӢ… (4un - un-1)

    3. Spacial Discretization:
        3.1 Diffusion (CDS):
            вҲӮВІu/вҲӮxВІ вүҲ (ui+1,j - 2ui,j + ui-1,j)/О”xВІ
            вҲӮВІu/вҲӮyВІ вүҲ (ui,j+1 - 2ui,j + ui,j-1)/О”yВІ
            вҶ’ AdE = AdW = ОјО”y/О”x
            вҶ’ AdN = AdS = ОјО”x/О”y
            вҶ’ AdP = AdE + AdW + AdN + AdS

        3.2 Convection (Upwind):
            a. Mass fluxes through faces:
                Ff = ПҒ(ufвӢ…nf)Af
            b. Upwind Scheme:
                Ff в©ҫ 0 вҶ’ uP, Ff < 0 вҶ’ uK, K = E,W,N,S
            вҶ’ AcE = max(-Fe, 0), AcW = max(Fw, 0), AcN = max(-Fn, 0), AcS = max(Fs, 0)
            вҶ’ AcP = AcE + AcW + AcN + AcS

    4.  Pressure gradient:
        x-momentum :
            - вҲ«вҲҮp dV вүҲ -О”V/О”x вӢ… (pE - pW)
            вҶ’ Qp_u = -О”y вӢ… (pE - pW)
        y-momentum :
            - вҲ«вҲҮp dV вүҲ -О”V/О”y вӢ… (pN - pS)
            вҶ’ Qp_v = -О”x вӢ… (pN - pS)
